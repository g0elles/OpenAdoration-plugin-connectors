# Security Policy

## Reporting a vulnerability

**Please do not open a public issue for security problems.**

Report privately via GitHub's **"Report a vulnerability"** button on the
**Security** tab, or email **ellesgabry5@gmail.com**. Include a description,
impact, reproduction steps, and the affected plugin/version.

You can expect an initial response within a few days.

## Scope notes

These plugins run **in-process at full trust** inside the OpenAdoration host and
fetch scripture text over the network. Reports about request handling, API-key
storage, and parsing of remote responses are especially welcome. API keys are
supplied by the user and are never committed to this repository.
